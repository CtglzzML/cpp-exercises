Este programa intenta abrir un archivo llamado file.txt usando std::ifstream.
Si el archivo no puede abrirse, se lanza una excepción std::runtime_error y se muestra un mensaje de error.
Si todo funciona correctamente, el programa indica que el archivo se abrió con éxito.

Contenido del ejemplo

Uso de try y catch

Comprobación de apertura de archivos con std::ifstream

Lanzamiento de excepciones con throw

Lectura del mensaje de error mediante e.what()

Objetivo

Demostrar de forma sencilla cómo manejar errores al abrir archivos en C++ usando excepciones.
